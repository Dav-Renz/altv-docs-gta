## Usage

There are several natives which allow you to read the hud color values and also modify them.\
To get the specific RGBA values of a hud color, you can use the native [getHudColour](https://natives.altv.mp/#/0x7C9C91AB74A0360F).\
Modification of the hud colors can be done with the natives [replaceHudColour](https://natives.altv.mp/#/0x1CCC708F0F850613) and [replaceHudColourWithRgba](https://natives.altv.mp/#/0xF314CF4F0211894E).

To use those colors in a drawn text, visit the article [text formatting](text-formatting.md).
